⚙ Параметры справедливого спреда к КС
Параметры расчёта
Форвардная кривая КС (из IRS KEYRATE)
«≤ год» — поток, гасящийся в пределах этого года, дисконтируется по указанной форвардной КС. Кривая влияет на результат; контрактный спред — только для подписи «контракт +N».
Как считается

Озон 1P-02 — флоатер: купон = КС + спред. Ищем постоянный спред s к КС, при котором приведённая стоимость потоков равна рыночной (грязной) цене. Это вменённая рынком кредитная премия (дисконт-маржа).

Live с MOEX: чистая цена, НКД, YTM, дата погашения, период купона, номинал.
Задаётся здесь: контрактный спред и форвардная кривая КС.

Алгоритм:
1. Даты купонов — шаг назад от погашения по периоду.
2. Для потока через t лет: купон от КС = фвдКС(t)·(дн/365)·номинал, дисконт-фактор (1+YTM)^t.
3. A = Σ(купон от КС)/df + номинал/df — это PV при спреде = 0;  B = Σ(дн/365·номинал)/df — чувствительность к спреду.
4. PV линейна по спреду, поэтому решаем напрямую: s = (грязная цена − A) / B.

Дисконтируем по рыночной YTM (конвенция расчёта коллег). Результат — в б.п. Это не Z-спред (тот — надбавка ко всей кривой ОФЗ, берётся готовым из MOEX).
